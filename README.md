# Enterprise SIEM Lab with Wazuh

## 🎯 Project Overview
Building a production-grade Security Information and Event Management (SIEM) lab using Wazuh to demonstrate SOC analyst capabilities including threat detection, log analysis, incident investigation, and security monitoring.

## 🏗️ Architecture
**Lab Environment:**
- Wazuh Manager (Ubuntu Server 22.04)
- Windows 10 Agent
- Ubuntu Desktop Agent
- Custom detection rules mapped to MITRE ATT&CK
